## Installation ##

1. Install Ultimate Addons for Beaver Builder (Lite) via the WordPress plugin directory or by uploading to wp-content/plugins.

2. After activating Ultimate Addons for Beaver Builder, go to Settings > Beaver Builder to configure the post types.

3. A Page Builder tab will appear in the Edit Page/Post area, and a Page Builder option will show in the WordPress Admin Bar on the front end.
